# Remote Controlled UGAV for Waste Disposal

## Overview
This project presents a Remote Controlled Unmanned Ground Aerial Vehicle (UGAV) designed for waste collection and disposal applications.

## Objective
To develop a remotely controlled vehicle capable of collecting and transporting waste safely.

## Components Used

- Mini Pix Flight Controller
- Li-Po Battery
- BLDC Motors and Propellers
- Electronic Speed Controller (ESC)
- DC Gear Motors
- L298N Motor Driver Module
- GPS Module
- FlySky FS-iA6B Receiver and Transmitter
- WeMos D1 Mini ESP8266
- Servo Motor Based Waste Disposal Mechanism

## Technologies

- Robotics and Automation
- Embedded Systems
- Wireless Communication
- GPS-Based Navigation
- Remote-Control Systems
- IoT Integration
  
## Applications
- Smart Waste Management
- Industrial Areas
- Hazardous Environments
