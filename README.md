# Remote Controlled UGAV for Waste Disposal

## Overview
This project presents a Remote Controlled Unmanned Ground Aerial Vehicle (UGAV) designed for waste collection and disposal applications.

## Objective
To develop a remotely controlled vehicle capable of collecting and transporting waste safely.

## Technologies Used
Components Used

- ESP32 Development Board
- IMU Sensor
- GPS Module
- Motor Driver Circuit
- DC Motors
- LiPo Battery
- Wireless Communication Module

Technologies

- Embedded Systems
- Robotics
- Sensor Integration
- Remote Control Systems
- Automation

## Applications
- Smart Waste Management
- Industrial Areas
- Hazardous Environments
